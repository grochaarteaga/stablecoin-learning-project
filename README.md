# Stablecoin Learning Project 

This repo documents my 14-day journey learning how stablecoins and crypto transactions work by building a **mini stablecoin wallet** with Python.

## Objectives
- Understand stablecoin mechanisms (USDC, DAI, USDT)
- Learn blockchain fundamentals
- Build and test real transactions on Ethereum / L2s
- Create a small wallet app in Python (web3.py)
- Optional: deploy a smart-contract escrow
